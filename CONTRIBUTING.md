# Contributing

Thank you for helping grow The Hidden Equations of Feeling.

## Project boundary

Every contribution must preserve the core boundary: these formulas are symbolic design patterns and creative metaphors. They are not clinical tools, diagnostic models, empirical laws, human scores, or instructions for manipulation.

## Formula proposal checklist

A new formula should include:

- Title
- Family
- Formula string
- Plain-language formula label for accessibility
- Variable list with symbols, meanings, and roles
- Tagline
- Derivation / rationale
- Cultural scope note
- Reflection prompt
- Example application
- Boundary note
- Misuse warning
- Cross-references

## Tone

Funny where it should be funny. Gentle where it touches pain. Claim-disciplined everywhere.
