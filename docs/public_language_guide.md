# Public Language Guide

This guide provides safe, ready-to-use language for describing **The Hidden Equations of Feeling** in public places such as the README, GitHub release notes, Zenodo, social posts, the app UI, and contributor docs.

## Core one-liner

> A symbolic formula atlas and interactive React app for mapping comedy, drama, emotion, and storytelling patterns as creative design equations.

## Short boundary line

> Symbolic design patterns only — not diagnosis, measurement, or universal law.

## Full boundary statement

> The Hidden Equations of Feeling is a symbolic formula atlas and creative design-pattern tool. These formulas are not clinical tools, diagnostic models, empirical laws, universal claims, or scores of a person.

## Project voice

Use language that feels:

- playful
- warm
- careful
- useful
- creative
- claim-disciplined
- emotionally respectful

Avoid language that feels:

- clinical without need
- manipulative
- shame-based
- over-scientific
- over-spiritualized
- fake-objective
- like a personality test
- like a diagnostic system

## Sensitive-topic language

For grief:

> Grief has no deadline. Use this card as a gentle reflection lens, not a schedule or solution.

For anger:

> Anger can point to harm, need, or boundary. This card is not a license to harm; it should route toward clarity, safety, and repair.

For fear:

> Fear formulas are reflection/story lenses. If danger is immediate, real-world safety comes first.

For shame:

> Shame cards must be handled with compassion. The goal is witness and repair, not exposure or punishment.

For love:

> Love formulas should never be used to excuse control, coercion, or harm. Love includes boundaries.
