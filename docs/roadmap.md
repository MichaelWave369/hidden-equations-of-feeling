# Roadmap

This roadmap keeps the project moving in small, safe releases.

## v0.3 — Public scaffold and publication foundation

Goal: make the project understandable, runnable, and ready for public review.

Status: in progress.

Includes:

- React/Vite app scaffold
- Formula card browser
- Search and family filtering
- Detail panel
- Compare mode
- Symbolic Formula Playground
- Formula data validator
- Notation key
- Variable glossary
- Expansion candidates
- GitHub Pages workflow
- Zenodo metadata
- Citation metadata
- Release checklist
- Setup guides
- Boundary and issue templates

Remaining v0.3 tasks:

- Import the full expanded formula-card dataset if not already live.
- Confirm GitHub Pages deploys successfully.
- Add DOI badge after Zenodo release.
- Improve README with screenshots or social card preview.
- Add a short demo GIF or image later if desired.

## v0.4 — Better app experience

Goal: turn the app from a basic atlas into a memorable interactive tool.

Candidate features:

- Formula of the Day
- Favorites saved locally
- Better compare mode with suggested pairings
- Shareable card URLs
- Copy formula as Markdown / plaintext / citation
- Printable formula-card view
- More polished Formula Playground
- Family landing pages
- Stronger mobile layout
- Accessibility pass for equations and controls

## v0.5 — Creator and educator tools

Goal: make the atlas useful in writing rooms, game design, teaching, and reflection.

Candidate features:

- Worked examples gallery
- NPC emotion design examples
- Comedy bit builder
- Drama scene builder
- Journaling prompt sequences
- Export selected cards to Markdown
- Export selected cards to printable PDF later
- Contributor review workflow

## v0.6 — Research and whitepaper polish

Goal: make the project stronger as a citable creative research artifact.

Candidate features:

- Expanded manuscript / whitepaper
- Cleaner references and related-work section
- Evaluation plan for writers and game designers
- Case studies
- Versioned formula changelog
- DOI release notes
- Stronger distinction between canonical cards and experimental candidates

## v1.0 — Stable public release

Goal: publish a stable, polished version that can be confidently shared.

Targets:

- Stable full formula dataset
- Green build and validation
- GitHub Pages live site
- Zenodo DOI
- Clear license structure
- Full README
- Whitepaper PDF or Markdown manuscript
- Accessible app flow
- Public contribution guide
- Boundary language visible everywhere

## Long-term ideas

- Card deck export
- Poster generator
- Obsidian vault export
- Notion template export
- Game/NPC design plugin
- Classroom activity pack
- Professor Phi guided walkthrough mode
- Local-first saved collections
- Formula remix sandbox

## Boundary lock

Every roadmap phase must preserve the core boundary:

> These are symbolic design equations and creative metaphors only. They are not clinical, diagnostic, empirical, universal, or human-score systems.
