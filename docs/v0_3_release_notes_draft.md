# v0.3.0 Release Notes Draft

## The Hidden Equations of Feeling v0.3.0

This release establishes the first public publication scaffold for **The Hidden Equations of Feeling**: a symbolic formula atlas and interactive React app for comedy, drama, emotion, storytelling, NPC/dialogue design, education, and reflective creative tools.

The project is designed to make invisible emotional and narrative patterns easier to see without reducing people to scores.

## Core boundary

> These formulas are symbolic design equations and creative metaphors only. They are not clinical tools, diagnostic models, empirical laws, universal claims, or scores of a person.

Short form:

> Pattern without shame. Wonder without overclaiming. Feeling without scoring.

## Intended users

This release is intended for:

- writers
- comedians
- game designers
- NPC/dialogue designers
- educators
- reflective-tool builders
- creative researchers
- people interested in playful emotional pattern language

## What this release is not

This release is not:

- therapy
- diagnosis
- psychological testing
- an emotional scoring system
- a scientific model of emotion
- a universal theory of human behavior
- a persuasion or manipulation tool

## Known limitations

- The formula dataset is still draft-stage.
- The React app is an early scaffold, not a polished final product.
- Some app features are planned but not complete, including shareable formula URLs, favorites, copy tools, and family landing pages.
- The manuscript is a working draft and needs expansion before formal publication.
- The project still needs public review, accessibility review, and broader contributor feedback.
