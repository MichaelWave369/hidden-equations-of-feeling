# The Hidden Equations of Feeling

A playful, claim-disciplined symbolic formula atlas for writers, game designers, NPC/dialogue designers, educators, and reflective creative tools.

**Boundary:** these formulas are creative metaphors and affective design patterns. They are not clinical tools, diagnostic models, empirical laws, or instructions for manipulating people.

## What is inside

- `src/data/formula_cards.json` — canonical source of truth for 52 v0.3 formula cards.
- `src/data/variable_glossary.json` — variable glossary with scoped-symbol notes.
- `src/data/formula_schema.json` — JSON Schema for formula cards.
- `docs/manuscript.md` — companion whitepaper skeleton for a citable Zenodo release.
- `docs/notation_key.md` — how to read the symbolic operators.
- `src/` — Vite + React starter app with cards, compare mode, and a symbolic Formula Playground.

## Quick start

```bash
npm install
npm run validate-data
npm run dev
npm run build
```
