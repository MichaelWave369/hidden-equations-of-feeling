# License Layout

This repository intentionally separates software code from creative content.

## Code

React, TypeScript, scripts, workflow configuration, and other software files are covered by `LICENSE-CODE` unless otherwise noted.

Recommended: MIT.

## Content

Formula text, formula JSON data, prose docs, examples, and manuscript materials are covered by `LICENSE-CONTENT` unless otherwise noted.

Recommended: CC BY 4.0.

## Art and generated images

Poster art, generated images, screenshots, and third-party homage/reference materials may use separate terms per release.

## Boundary

The license allows creative reuse, but it does not turn these symbolic formulas into clinical, diagnostic, empirical, or manipulative tools.
